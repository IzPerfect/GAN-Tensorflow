Generative Adversarial Networks(GAN)
===
Implementation of various GANs using Tensorflow on MNIST data

- Vanilla GAN
- LSGAN(Least Square)
- CGAN(Conditional)

Requirement
---
1. Tensorflow 1.4.0
2. Python packages : numpy, matplotlib, os, tensorflow
